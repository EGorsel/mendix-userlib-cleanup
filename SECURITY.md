# Security Policy

## Supported Versions

We are committed to providing a secure cleanup suite for the Mendix community. The following versions of the tool are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v4.0.0+ | :white_check_mark: |
| < v4.0.0| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately.

**Do not open a public issue.** Instead, please follow these steps:

1. **Email**: Send a detailed report to the project maintainer (Erik van Gorsel).
2. **Details**: Include a description of the vulnerability, the steps to reproduce it, and the potential impact.
3. **Acknowledgment**: We will acknowledge receipt of your report and provide a timeline for resolution.

Thank you for helping keep the Mendix community safe!
